### 上手指南

###### 开发前的配置要求

1. python 3.11
2. windows系统

###### **安装步骤**

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo

```sh
git clone https://github.com/Meta217/Application-of-ARIMA-in-Volcanic-Earthquakes.git
```
